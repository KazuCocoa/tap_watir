# TapWatir

Watir for testing your Mobile Devices. Powered by Appium.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'tap_watir'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install tap_watir
